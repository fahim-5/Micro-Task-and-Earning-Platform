# Micro-Task and Earning Platform (Frontend)

This is the frontend for the Micro-Task and Earning Platform — a MERN-based application where Buyers can post micro-tasks and Workers can complete them to earn coins.

Run locally:

```bash
cd frontend
npm install
npm run dev
```

Environment:

- Use `.env` or `VITE_` variables to configure API base URLs and Firebase keys if needed.

See the project root README for full setup and deployment instructions.
